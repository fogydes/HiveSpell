# Theme Assets

This folder is reserved for future theme-specific assets.

Suggested structure:

- `public/themes/<theme-id>/cursor.svg`
- `public/themes/<theme-id>/hero-pattern.svg`
- `public/themes/<theme-id>/overlay-pattern.svg`
- `public/themes/<theme-id>/panel-frame.svg`

The theme package contract lives in [`../../data/themePackages.ts`](../../data/themePackages.ts).
